Place real images in this folder to replace the placeholders:

- logo.png — used in the header (recommended 128x128)
- hoodies.png — used for the Hoodies collection (recommended 800x800)
- tshirt.png — used for the T-shirt collection (recommended 800x800)

After replacing files run:

```powershell
flutter pub get
flutter clean
flutter run -d chrome
```

Use hot restart if the app is already running.
